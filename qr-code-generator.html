How this QR code generator works

A QR code stores data — most often a URL, but also plain text, Wi-Fi credentials, or contact details — as a pattern of black-and-white squares that any phone camera can scan. This tool encodes whatever you type into that pattern locally in your browser and renders it as an image you can download and use anywhere: business cards, posters, packaging, presentation slides, or a link on a web page.

Error correction controls how much of the code can be damaged, obscured, or misprinted and still scan successfully — useful if you're printing small, on a textured surface, or placing a logo in the middle of the code. Higher levels make the pattern denser (which can make very short text harder to read at small sizes), so M is a reasonable default unless you have a specific reason to go higher.

Nothing you type is sent to a server — the QR code is generated and rendered entirely in your browser, so it's safe to use for private links, internal Wi-Fi passwords, or anything else you wouldn't want passing through a third party.
