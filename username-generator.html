Random Name & Username Generator

What this generator is useful for

Username mode combines a random adjective, a random noun, and a three-digit number (like CrimsonFalcon482) — the same pattern used by countless gamertag and account-name generators, useful for placeholder accounts, test fixtures, or picking a new handle without staring at a blank field. Display name mode pairs random first and last names for more realistic-looking sample data, handy when populating a demo, mockup, or test database that needs to look like real users without using anyone's actual name.

Everything is generated locally with no server round-trip, so you can generate as many as you need without rate limits.
