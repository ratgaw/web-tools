How PDF merging works here

This tool uses pdf-lib, an open-source library, to copy every page from each selected PDF into one new document, in the order you arrange them. Common uses: combining scanned pages into a single file, stitching together chapters or sections that were exported separately, or assembling a set of reports into one document to send.

Because it runs client-side, there's no file-size limit imposed by a server upload, and your documents — which might contain sensitive contracts, records, or personal information — never leave your device.
