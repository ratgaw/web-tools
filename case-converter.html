When to use each case

camelCase and snake_case are the two most common variable-naming conventions in code — camelCase in JavaScript, Java, and C-family languages, snake_case in Python, Ruby, and most database column names. kebab-case is standard for URLs, CSS class names, and HTML attributes. Title Case and Sentence case are for prose — headings and titles versus regular sentences. UPPERCASE and lowercase cover the simple cases, like fixing text someone sent with caps lock on.
